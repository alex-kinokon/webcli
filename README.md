# webcli

A simple cli to show a web page in a window.

## Help

```
USAGE: ./web <target> [--width <width>] [--height <height>] [--min-width <min-width>] [--min-height <min-height>] [--max-width <max-width>] [--max-height <max-height>] [--dev-tools]

ARGUMENTS:
  <target>                URL or HTML file

OPTIONS:
  -w, --width <width>     Window width
  -h, --height <height>   Window height
  --min-width <min-width> Minimum window width, cannot be used with --width
  --min-height <min-height>
                          Minimum window height, cannot be used with --height
  --max-width <max-width> Maximum window width, cannot be used with --width
  --max-height <max-height>
                          Maximum window height, cannot be used with --height
  -d, --dev-tools         Allow developer tools
  -h, --help              Show help information.

```
